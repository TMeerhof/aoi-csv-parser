How does it work

Make a inventory of your collection in excel or simmilar software.
Save as a .CSV(comma seperated value) file. Uplaod here and download the XML file.

Rules:

First colum MUST be a unique identiefier for your item. fieldname should be UniqueIdentifier
The rest of the colums must be one of the following:
  • setSpec
  • title
  • creator
  • subject
  • description
  • contribitor
  • publisher
  • date
  • type
  • format
  • identifier
  • source
  • language
  • relation
  • coverage
  • rights
You can use the same field multiple times, so you can have multiple colums called subject, if youre items cover multiple subjects.
It is not adviced to have multiple use the fields title and discription multiple times.
